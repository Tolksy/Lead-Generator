# Harbour City Car Leads

Single-page marketing site designed to capture high-intent car buyer leads across Nanaimo, BC and the Mid-Island corridor.

## Tech stack
- Plain HTML5, CSS3, and vanilla JavaScript
- Responsive layout with CSS Grid/Flexbox
- Client-side form handling stub (`script.js`) for integrating with lead APIs or automation platforms

## Local development
Open `index.html` in any modern browser. No build step is required.

## Customization points
- Update imagery by editing the `background` definitions inside `styles.css`
- Wire the intake form to your CRM or webhook inside `script.js`
- Modify copy to highlight new incentives, partner lots, or neighbourhoods

## Deployment
Drag-and-drop the project into any static hosting service (Netlify, Vercel, S3, etc.).
